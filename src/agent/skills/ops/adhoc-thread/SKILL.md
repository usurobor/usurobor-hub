# Adhoc Thread

Create ad-hoc threads for discussions, proposals, learnings.

## When

| Trigger | Example |
|---------|---------|
| Proposal | Architecture change |
| Learning | Mistake captured |
| Question | Need peer input |
| Decision | Why X over Y |

## Location

```
threads/adhoc/YYYYMMDD-topic.md
```

## Structure

```markdown
# [Title]

**Date:** YYYY-MM-DD
**From:** [author]
**To:** [audience]
**Type:** [proposal | learning | question | decision]

---

[Content]

---

**Status:** [Open | Resolved]
```

## Types

### Proposal (CLP)

```markdown
## TERMS
## POINTER
## EXIT
```

### Learning

```markdown
## What Happened
## Why It Was Wrong
## Lesson
```

### Question

```markdown
## Context
## Question
## Options
```

### Decision

```markdown
## Decision
## Alternatives
## Why This One
```

## Lifecycle

Create → Share → Discuss → Resolve
